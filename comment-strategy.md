# 1. 🎯 1차 접근 (댓글용 — 가장 중요)

👉 목적: **경계심 낮추고 DM 유도**

```text
안녕하세요, 영상 잘 봤습니다.

혹시 해당 블랙박스 영상 관련해서 간단히 문의드리고 싶은 게 있어서요.
괜찮으시면 쪽지나 댓글 확인 부탁드립니다!
```

✔ 포인트

* “돈” 언급 ❌
* “플랫폼” 언급 ❌
  → 자연스럽게 접근

---

# 2. 📩 2차 접근 (쪽지/DM)

👉 목적: **거래 전환**

```text
안녕하세요!

현재 블랙박스 영상 기반 콘텐츠 플랫폼을 준비 중인데,
작성자님 영상이 적합할 것 같아 연락드렸습니다.

영상은 개인정보(얼굴, 번호판 등) 모두 블러 처리 후
플랫폼 내 콘텐츠로 활용될 예정이며,
소정의 보상도 함께 드리고 있습니다.

가능하시다면 간단히 안내드리고 싶습니다 :)
```

✔ 포인트

* “안전성 + 보상” 동시에 강조

# 3. 💰 3차 (가격 제시 + 조건)

👉 목적: **결정 유도**

```text
감사합니다!

해당 영상은 약 5,000원 ~ 10,000원 수준으로 보상 가능하며,
플랫폼 내 게시 및 일부 편집(블러 처리 포함) 후 활용될 예정입니다.

진행 시 간단한 동의 절차 후
영상 전달 → 바로 지급 진행드립니다.
```

```text
진행 가능하시면 아래 내용 확인 후 동의 부탁드립니다.

[콘텐츠 이용 동의]

본인은 본인이 촬영한 영상을 제공하며,
플랫폼 내 게시 및 상업적 이용, 편집에 동의합니다.

또한 제3자의 권리를 침해하지 않았음을 보증합니다.

(이름 / 날짜)
```
